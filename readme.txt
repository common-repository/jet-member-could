== Installation ==

1. Upload `j-rnd-members.php` to the `/wp-content/plugins/` directory
2. Upload 'goall.png' to the 'images' directory your theme (or other)
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add widget to site
5. Set Title, Count and Image Path (eg: /wp-content )
